Whitepaper USDt.z

Technická dokumentace a API

Smart kontrakt USDt.z (BEP-20) – adresa 0x913ef7244bEd019b95Ec5321408eE4CE83E25c01 na Binance Smart Chain (BSC).

Podporované operace:

  • mint(address recipient, uint256 amount)
  • burn(uint256 amount)
  • transfer(address to, uint256 amount)
  • kolaterální operace pro zajištění půjček a swapů

API / Web3 interakce:

  • Čtení stavu: balanceOf(address), totalSupply(), allowance(owner, spender)
  • Zápis operací: mint, burn, transfer přes wallet s přístupem vlastníka
  • Kolaterální protokoly: depositCollateral(), withdrawCollateral(), swapCollateral()

Podmínky obchodních operací

  • Zajištěná mimoburzovní půjčka – rychlost 1–3 dny, ochrana kapitálu 100%, RPSN 6–12 %, doba splatnosti 30–90 dní.
  • Časově vázaný swap stablecoinů – rychlost tentýž den, poplatek 0,5–2 % měsíčně.
  • Výnos k hotovosti – rychlé nastavení, 5–15 % RPY, variabilní riziko.
  • Tokenizované dluhopisy – rychlost 1–2 týdny, fixní kupón 6–10 %, splatnost 90–180 dní.
  • Repo dohoda s odkupem – rychlost do 48 hodin, poplatek 0,5–1,5 % měsíčně.
